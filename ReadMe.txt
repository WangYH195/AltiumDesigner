=====HERO公有库 Copr. HITWH_HERO=====
=========仅供HERO团队使用==========

本公有库分为三个部分：
封装库
模版库
电路模块库

注意：
在对本库进行编辑前务必知悉以下内容！
为避免GitHub版本冲突，在编辑前需新建txt文件，更改文件名及文件类型后缀与原文件一致（此文件名称为“编辑状态标识”），并将编辑状态标识改为以下格式：“被编辑文件名”_“编辑人”_“开始编辑时间”，在编辑完成后，需将编辑状态标识清除。

e.g.    
被编辑文件：Proj.LIB
编辑中：Proj_王宇航_2019年2月28日.LIB  （中文输入法输入“riqi”，一般会自动候选“2019年3月1日”）
未编辑：Proj_.LIB 

被编辑文件夹：HERO
编辑中：HERO_王宇航_2019年2月28日
未编辑：HERO_

任何文件同时只允许一人进行编辑！！！
对任何文件编辑前，需先PULL版本库，后方可打开文件，以避免本地文件版本过期。
编辑完成后，在Commit框记录总结和编辑内容，及时修改状态标识，并将已完成编辑的文件PUSH到GitHub，以此来减少个人的编辑占用时长。
如需对整个文件夹进行编辑，标准与单文件相同。

.gitignore文件为忽略控制文件，无需编辑，切勿删除！！！
